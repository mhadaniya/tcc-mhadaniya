\chapter{Conclusão}

		Como foi apresentado neste trabalho, muitas áreas de estudos têm se esforçado para achar um equilibrio para a expansão desenfreada da informação e a qualidade.
		
		Com os conceitos básicos sobre Recuperação de Informação, temos uma visão clara de que categorizar os documentos não é algo trivial, principalmente hoje, quando tomamos a \emph{Web} como base de documentos, e encontramos um dominio caótico quanto a organização e estruturação documental. Porém, é uma área mais madura pela simples razão de existir a mais tempo.
		
		Os conceitos de Extração de Informação sempre estiveram presentes, pois além de recuperar os documentos corretamente, é necessário fazer uso das informações contidas no documento. Para tal tarefa, nasceu a Extração de Informação que através de muitas técnicas tenta amortizar os resultados perto de um ideal. Muitas técnicas se misturam para tal objetivo.
		
		Com todos estes conceitos usados em conjunto, temos o conceito de Web Mining, que simplesmente é a utilização das tecnicas de Mineração de Dados clássica contemplando os documentos da Web buscando descobrir e extrair padrões automaticamente. Como a \emph{Web} é uma fonte de dados heterogenea, temos abordagens que se diferenciam de acordo com o foco estrutural ou de conteúdo.
		
		Finalizando, temos a revolução da própria \emph{Web} que busca uma organização própria para que seres humanos e também máquinas possam processar suas informações, nos introduzindo a Web Semântica. Com o conceito de Web Semântica, temos uma \emph{Web} com um paradigma de interação entre usuário e \emph{Web} completamente novo.
		
		Os trabalhos atualmente estão voltados na realização da Web Semântica, buscando torna-la uma realidade da maneira como foi preconizada. Muitas discussões e pequisas são feitas na definição de ontologias, de uma maneira que elas se tornem um padrão aceito mundialmente.
